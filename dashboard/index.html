Routing accuracy competition. Beat the current best, earn TAO via Gittensor.

🏆 How to compete

  1. Train a routing head: python -m trinity.train --benchmark math500
  2. Pack your submission: python scripts/pack_submission.py --run-dir ... --miner-name you
  3. Submit a PR to this repo with your head weights + receipt
  4. Win: if your head beats the current best, your PR is merged → TAO
